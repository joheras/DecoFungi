Tutorial page

1. Images

This web application analyse two types of fungi images, all of them contained in their respective Petry dish.

  • Fungi image: that correspond to the dyed fungi, is the image that we want to obtain the prediction.
  • Control image: is only dye. We use it to improve the reliability of the prediction.

2. Analisys

We can group the methods of analysis according to two criteria:

  • Analyze one image or a zip folder of images.
  • Analysis using or not control image.

One image Zip file
Without control image Analyze one image Analyze a zip file
Using control image Analyze one image with control one Analyze a zip with control image

2.1 Steps for analysis:

  • Step 1: Select the analisys method option on the Menu page.

    Then it will appear the correspond charge image/zip formulary.
  • Step 2:Pressing 'Examinar' button apeared the following window. Choose the corresponding file to analyse that it must be in the directory media of our application.

    Take care it must be an image or a zip file extension.

  • Step 3: Once the file is selected, click on the "Upload and analyze" button to get the prediction.

    • 3. Prediction page

    3.1 Prediction aspects:The prediction has the following form.

    The prediction value and the sample name are given by the model, while the dye used field is empty and we must to introduce it.
    Note that the prediction, name and dye data are obligatory. Also we can introduce some notes optionally.

    If we are not satisfied with the data obtained by the application (prediction and name), we can change them.
    To change the name we just have to press the arrow on the combobox. Then it will appear the possible predictive values.

    To change the sample name we must to click on the pencil icon, then we can write into the textbox area.

    Once we have clicked on the red textbox background color disappeared.

    3.2 Button menu:

    • Back to analyse: Back to the previous page, corresponding to the file charge formulary.
    • Restore: If we want to return to the initial state just after obtaining the prediction, we have the restore option which allows to return to the initial values.
    • Save data/ Save selection: Save the obtained and introduced data into a Data base.

    3.3 Zip funcionality
    In the case of using a method that analyzes a zip file we can select those predictions that wanted to be saved. To do this, in the upper right of each prediction a checkbox appears. If the checkbox are checked it means that the correspond prediction is going to be saved.

    Also we have a checkbox called "Select All:" that allows you to select and deselect all predictions.
