Welcome to the web application for automatic characterisation of dye decolorisation in fungal strains. In this application, you can upload and analyse images from your Petry dish and get the corresponding prediction. Through the following menu you can access to the different prediction options, which are the following:

  • Upload and analyse a single image of a fungal strain.
  • Upload and analyse two images, one of them corresponding to the image of a fungal strain, and the other corresponds to the control image.
  • Upload and analyse a zip file which contains a group of fungal strains.
  • Upload and analyse a zip file which contains a group of fungal strains and a control image (the control image must be the same for all the fungal strains of the zip).
